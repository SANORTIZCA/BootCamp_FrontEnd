PROPIEDADES DE FUENTE

Propiedad Descripción
font-family Determina uno o más nombres de fuentes o familias de fuentes. Se usan en orden
font-style Determina el estilo de la letra
font-size Determina el tamaño de la fuente

TEXTOS Y PÁRRAFOS

Propiedad Descripción
color Determina el color del texto
line-height Determina el espacio entre las líneas
text-align Determina la alineación del texto
width Determina el ancho de un texto o una imagen
height Determina la altura de un texto o una imagen
text-decoration Especifica la decoración añadida al texto.

COLORES DE FONDO

Propiedad Descripción
background-color Determina el color de fondo
background-repeat Determina cómo se repite la imagen de fondo
background-position Determina donde posiciones la imagen con respecto a la esquina superior izquierda
background-attachment Determina si la imagen de fondo se quedara en el lugar de la pantalal cuando esta se desplace

MÁRGENES

Propiedad Descripción
margin Permite el acceso a las propiedades de márgenes
margin-top Valor para el margen superior
margin-right Valor del margen derecho
margin-bottom Valor del margen inferior
margin-left Valor del margen izquierdo

BORDES

Propiedad Descripción
border Permite el acceso a las propiedades de bordes
border-collapse Permite agregar o eliminar el efecto "3D"
border-radius Agrega efectos de sombra alrededor del marco de un elemento

RELLENO

Propiedad Descripción
padding Permite el acceso a las propiedades de relleno
padding-top Relleno entre el elemento y el borde superior
padding-right Relleno entre el elemento y el borde derecho

TABLAS

Propiedad Descripción
border-collapse Permite combinar los bordes de las celdas (collapse), no los combina (separate)
border-spacing Da el espacio de las celdas
table-layout Da un ancho fijo o variable a la celda

CAJAS

Propiedad Descripción
background-color Da color de fondo del contenido y del relleno
color Da color del contenido del elemento (generalmente texto)
text-shadow Da coloca una sombra difuminada en el texto dentro del elemento
display selecciona el modo de visualización para el elemento
box-shadow Agrega efectos de sombra alrededor del marco de un elemento
cursor Establece el cursor del mouse, si lo hay, para mostrar cuando el puntero del mouse está sobre un elemento.
transition Es una propiedad abreviada de propiedad de transición, duración de transición, función de temporización de transición y retraso de transición.
:hover Sucede cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Por lo general, se activa cuando el usuario pasa el cursor sobre un elemento (puntero del mouse).
