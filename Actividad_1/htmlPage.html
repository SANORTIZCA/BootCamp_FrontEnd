ETIQUETAS PARA LA BASE

Son las primeras etiquetas que usas al crear tu archivo HTML. Necesarias para casi cualquier proyecto.

Etiqueta Descripción Propiedad Descripción
<!DOCTYPE html> Indica la versión de HTML
<html> Es la raiz del archivo
<head> Engloba los metadatos
<title> Es el título en la pestaña del navegador
<link> Enlaza archivo CSS rel="" Para importar una hoja de estilo externa
<style> Es la raiz del archivo
<script> Enlaza o escribe JavaScript en el documento HTML
<metat> Características sobre la página charset="" Para que el navegador reconozca los caracteres que se van a usar.
<body> Cuerpo / Elementos visibles de la página

ETIQUETAS DE TEXTO

Se usa para cuando se desea darle funciones, jerarquías y estilos diferentes a el texto

Etiqueta Descripción Propiedad Descripción
<p> Crea párrafos
<h1><h2><h3><h4><h5><h6> Para dar estructuraa los títulos y subtítulos, h1 es el más relevante
<b> Texto en negrilla
<i> Texto en cursiva
<u> Texto subrayado
<a> Para agregar un link target="" Especifica donde se abre el documento vinculado
<span> Abarcar. Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.
<strong> Texto importanta de la página, resalta como negrilla
<em> Marca con énfasis las partes importantes de un texto, da cursiva
<blockquote> Texto en forma de cita
<br> Crea un salto de línea

ETIQUETAS DE MULTIMEDIA

Se usa para insertar contenidos de otras páginas web

Etiqueta Descripción Propiedad Descripción
<img> Se usa para insertar una imagen src="" El atributo src requerido especifica la URL de la imagen.
<video> Se usa para insertar un video
<iframe> Se usar para insertar contenido de otras páginas
<audio> Se usa para insertar audio
<source> especifica recursos de medios múltiples para las etiquetas <picture>, <audio>, o <video>.

ETIQUETAS PARA ESTRUCTURAR EL CONTENIDO

Si se desea dar orden al texto, las imágenes y todo lo que pongamos en la página web

Etiqueta Descripción
<ul> Crea una lista des-ordenada
<ol> Crea una lista ordenada
<li> Define los elementos de las listas
<hr> Rompe dos secciones de la web, usada como separador
<div> Secciona y/o agrupa etiquetas
<nav> Crea el menú de navegación
<header> Define la cabecera de la página
<footer> Define el píe de la página

ETIQUETAS PARA CREAR TABLAS

Se usa para dar orden y formato a los datos

Etiqueta Descripción
<table> Crea una tabla
<tr> Crea las filas
<td> Crea las columnas
<th> Crea el título o encabezado de cada columna
<col> Especifica las propiedades de las columnas

ETIQUETAS PARA CREAR FORMULARIOS

Se usa para la creación de formularios y recopilar información del usuario

Etiqueta Descripción
<form> Crea un formulario
<input> Campo para introducir datos
<label> Para dar el titulo al input
<textarea> Campo para introducir bastante texto
<button> Crea un botón
